DESCRIPTION:
A booking website for African braid styles, natural hair care, 
and wig installations. Promotes inclusivity by welcoming all genders.
The system allows clients to book appointments with professional stylists,
browse hairstyle options, and manage their bookings online.

FEATURES:
- User registration and login system
- Hairstyle gallery with categorized images
- Stylist directory with professional profiles
- Online booking system with appointment management
- Admin panel for managing content and users
- MySQL database for data storage

TECHNOLOGIES:
Frontend:HTML5, CSS3, JavaScript
Backend:PHP
Database: MySQL (via XAMPP)
Server:Apache
Development: XAMPP Control Panel


DATABASE SETUP:
1. In phpMyAdmin
2. Create database: `hair_booking`
